# Spotlight_Frontend

## Setup
* node version: `11.1.0`

## Development
* Run develop mode locally
  ```
  npm run dev
  ```
  and then open `localhost:3000`
  
* Run production mode locally
  ```
  npm run prod
  ```
  and then open `localhost:5000`
